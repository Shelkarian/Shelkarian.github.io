# Кулачный бой
> Владение рукопашным боем.

## Что даёт

Усиливает удары без оружия: урон в рукопашной растёт с прогрессом навыка.

## Как учится

В бою — дерясь без оружия (книг нет). Обучение быстрее по живым целям.
