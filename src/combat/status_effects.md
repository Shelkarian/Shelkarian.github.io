# Эффекты и состояния

## 1. Концепция
Состояния показывают, что с существом происходит после урона, болезней, магии, голода или окружения.

---

## 2. Базовые состояния
| Состояние | Эффект |
| :--- | :--- |
| Кровотечение | Постепенная потеря крови. |
| Оглушение | Временная потеря контроля. |
| Замедление | Снижение скорости движения. |
| Горение | Периодический урон и паника. |
| Отравление | Постепенный урон или штрафы. |
| Страх | Ошибки действий, дрожь, желание отступить. |

## 3. Правила
1. Состояние должно иметь понятную причину.
2. Игрок должен видеть, что состояние активно.
3. У состояния должен быть способ лечения или окончания.

## 4. Открытые вопросы
1. Складываются ли одинаковые состояния?
2. Есть ли иммунитет после окончания эффекта?
3. Какие состояния видны другим игрокам?
