# Дальний бой

## 1. Концепция
Дальний бой должен быть сильным при подготовке, но слабым в панике и ближней свалке.

---

## 2. Игровой цикл
1. Игрок берет оружие дальнего боя.
2. Проверяет боеприпасы.
3. Натягивает/заряжает оружие.
4. Целится.
5. Стреляет.
6. Система проверяет точность, укрытие, броню и дистанцию.

## 3. Оружие
| Оружие | Особенность |
| :--- | :--- |
| Лук | Быстрее, тише, слабее против брони. |
| Арбалет | Дольше заряжается, сильнее пробивает. |
| Метательное | Быстро используется, ограничено запасом. |

## 4. Открытые вопросы
1. Нужна ли отдельная механика прицеливания?
2. Как сильно движение снижает точность?
3. Можно ли подбирать стрелы после боя?
