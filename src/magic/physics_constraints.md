# Магическая физика

## 1. Концепция
Магическая физика описывает, как заклинания взаимодействуют с пространством, предметами, жидкостями, огнем и существами.

## 2. Правила
| Ситуация | Поведение |
| :--- | :--- |
| Стена между магом и целью | Заклинание блокируется или требует линии видимости. |
| Вода и молния | Может усиливать или распространять эффект. |
| Огонь и дождь | Дождь ослабляет открытое пламя. |
| Тяжелый предмет | Сложнее двигать магией. |

## 3. Открытые вопросы
1. Нужна ли линия видимости для всех заклинаний?
2. Может ли магия ломать стены?
3. Какие эффекты остаются в мире после каста?
