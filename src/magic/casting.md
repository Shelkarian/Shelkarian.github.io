# Система маны

## 1. Концепция
Мана ограничивает частоту применения магии и отделяет обычное действие от сверхъестественного.

## 2. Правила
| Параметр | Описание |
| :--- | :--- |
| Запас | Сколько магии персонаж может потратить. |
| Восстановление | Как быстро ресурс возвращается. |
| Перегруз | Что происходит при попытке колдовать сверх лимита. |
| Фокус | Предмет или состояние, улучшающее каст. |

## 3. Открытые вопросы
1. Мана общая для всех школ или у школ свои ресурсы?
2. Можно ли тратить здоровье вместо маны?
3. Как игрок видит нехватку ресурса?
