# Ритуалы

## 1. Концепция
Ритуалы — медленная магия с подготовкой, местом, предметами и участниками.

## 2. Игровой цикл
1. Игрок выбирает ритуал.
2. Подготавливает место.
3. Раскладывает компоненты.
4. Выполняет действия в нужном порядке.
5. Получает эффект, сбой или последствие.

## 3. Компоненты
| Компонент | Пример |
| :--- | :--- |
| Место | Алтарь, круг, руины. |
| Предмет | Свеча, кровь, камень, книга. |
| Время | Ночь, буря, определенная фаза. |
| Участник | Жрец, маг, жертва, свидетель. |
