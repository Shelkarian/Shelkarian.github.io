# Магия в бою

## 1. Концепция
Боевая магия дает контроль, урон, защиту и изменение поля боя, но требует времени, ресурса и риска.

## 2. Типы заклинаний
| Тип | Роль |
| :--- | :--- |
| Урон | Прямое поражение цели. |
| Контроль | Замедление, страх, удержание, ослепление. |
| Защита | Барьер, сопротивление, снятие эффекта. |
| Зона | Огонь, туман, лед, проклятая область. |
