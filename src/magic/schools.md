# Школы магии

## 1. Концепция
Школы магии разделяют заклинания по стилю, источнику и игровым задачам.

## 2. Черновые школы
| Школа | Роль |
| :--- | :--- |
| Огонь | Урон, свет, поджог. |
| Земля | Защита, стены, тяжесть. |
| Кровь | Риск, жертва, лечение через вред. |
| Тень | Скрытность, страх, ослабление. |
| Свет | Защита, очищение, поддержка. |
