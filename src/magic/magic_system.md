# Система магии

## 1. Концепция
Магия — отдельный слой действий, который работает через школы, ресурс, условия каста и последствия.

## 2. Игровой цикл
1. Игрок получает доступ к школе или заклинанию.
2. Проверяет ресурс и условия.
3. Начинает каст.
4. Система проверяет прерывание, цель и окружение.
5. Заклинание срабатывает или дает сбой.

## 3. Ограничители
| Ограничитель | Зачем нужен |
| :--- | :--- |
| Мана | Не дает спамить. |
| Время каста | Позволяет прервать. |
| Компоненты | Привязывает магию к экономике. |
| Риск | Делает сильные эффекты опасными. |
