# Освещение и аудио

## 1. Концепция
Свет и звук должны помогать читать ситуацию: где опасно, где кто-то идет, где работает станция или горит огонь.

## 2. Свет
| Источник | Роль |
| :--- | :--- |
| Факел | Переносной слабый свет. |
| Костер | Свет и тепло вокруг лагеря. |
| Лампа | Стабильное освещение помещений. |
| Магический свет | Особый источник, если нужен системе магии. |

## 3. Звук
| Событие | Звук |
| :--- | :--- |
| Шаги | Выдают движение рядом. |
| Работа станции | Показывает процесс без открытия UI. |
| Удар | Подсказывает тип столкновения. |
| Опасность | Дает сигнал игроку до прямого урона. |
