# Уголь
> Топливо для плавки металла.

## Что это

Сырьё `MedievalRawCoal`. Само по себе не перерабатывается — расходуется как топливо.

## Откуда берётся

Добыча угольных залежей (`MedievalCoalOre`).

## Где используется

[Плавильня](../lathe/smelter.md): топливо для переплавки руды в слитки (например, +100 угля на
слиток железа/стали).
