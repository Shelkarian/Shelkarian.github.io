# Распределение ресурсов

## 1. Концепция
Ресурсы распределяются так, чтобы игроки двигались по миру, торговали и конфликтовали за важные места.

## 2. Типы ресурсов
| Ресурс | Где искать |
| :--- | :--- |
| Дерево | Лес, посадки, старые постройки. |
| Камень | Карьеры, горы, руины. |
| Руда | Шахты, жилы, опасные зоны. |
| Травы | Поля, болота, лес. |
| Еда | Фермы, охота, рыбалка, торговля. |
