# Зонирование и доступ

## 1. Концепция
Зонирование делит мир на общественные, рабочие, опасные, частные и запретные области.

## 2. Типы зон
| Зона | Пример |
| :--- | :--- |
| Общественная | Площадь, рынок, трактир. |
| Рабочая | Кузня, кухня, склад. |
| Частная | Дом, комната, лавка. |
| Опасная | Шахта, руины, лес ночью. |
| Запретная | Тюрьма, архив, алтарь, хранилище. |
