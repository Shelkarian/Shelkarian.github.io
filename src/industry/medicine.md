# Медицина

## 1. Концепция
Медицина в ремесле отвечает за производство лекарств, бинтов, мазей, настоек и хирургических инструментов.

## 2. Производственный цикл
1. Игрок добывает травы, жир, спирт, ткань или реагенты.
2. Подготавливает сырье.
3. Создает лекарственную форму.
4. Проверяет качество.
5. Передает предмет в систему лечения.

## 3. Изделия
| Предмет | Назначение |
| :--- | :--- |
| Бинт | Остановка кровотечения. |
| Мазь | Лечение ожогов и ран. |
| Настойка | Внутренний эффект, риск передозировки. |
| Шина | Фиксация поврежденной конечности. |
