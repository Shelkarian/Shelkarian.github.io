
## 1. Концепция
Система полного цикла земледелия, основанная на физическом взаимодействии с окружением и менеджменте ресурсов почвы (система NPK — Азот, Фосфор, Калий). Растения делятся на выращиваемые (требуют грядок и истощают почву) и дикорастущие (собираются в мире).

---

## 2. Игровой цикл

### Этап 1: Подготовка почвы
*Создание пригодной среды для посева.*

1.  Игрок держит **земледельческий инструмент** (мотыга/лопата) в активной руке
2.  Клик инструментом по подходящему тайлу земли (грунт, грязь)
3.  Появляется короткий прогресс-бар
4.  **Результат:**
    * Тайл превращается во вспаханную грядку
    * Грядка получает базовый запас минералов (N, P, K) и уровень влажности 0.

### Этап 2: Посадка
*Внесение семян и корректировка химического состава почвы.*

1.  **Удобрение (опционально):** Игрок применяет реагент (зола, костная мука, селитра. Или алхимический прикол) на грядку. Почва поглощает минералы. 
2.  **Посадка:** Игрок кликает семенами по вспаханной грядке.
**Условия:** Если уровень освещения не соответствует требованиям семени (например, злакам нужен свет), семя не прорастает или получает дебафф к скорости роста.

### Этап 3: Рост

1.  **Потребление (тик роста):** На каждом этапе роста растение забирает из почвы нужное количество влаги и специфичных минералов (N, P, K).
Если нужного минерала или воды нет, растение теряет здоровье.
   **Угрозы: болезнь растений, вредители, скверна, аномалии** 

### Этап 4: Сбор урожая
*Получение конечного продукта.*

1.  Растение достигает финальной стадии.
2.  Игрок использует руки или **серп, косу**.
3.  **Результат:**
    * Растение исчезает
    * Появляются продукты (овощи, злаки, травы)
    * Грядка остается вспаханной, но с измененным пулом минералов.

#### Формула качества урожая
`Урожайность = Базовое значение + (Здоровье растения * Коэффициент почвы)`

| Состояние грядки | Результат | Пример |
| :--- | :--- | :--- |
| **Идеальное (Все NPK в норме, полито)** | **Богатый урожай** | Максимум плодов, шанс **мутации** |
| **Норма (Небольшой дефицит)** | **Норма** | Стандартное количество плодов |
| **Истощение (Нет NPK или воды)** | **Бедный урожай** | 1 плод, низкого качества |
| **Больное** | **Брак** | Гнилая биомасса, мертвые корни. |

## 3. Расширенный Аудио-дизайн (SFX)

### А. Динамика звуков земледелия
| Процесс | Стадия / Триггер | Тип звука | Описание (Референс) |
| :--- | :--- | :--- | :--- |
| **Вспашка** | **Удар мотыгой** | Once | Глухой звук удара металла о плотную землю, хруст корней. |
| **Посадка** | **Семя в грунт** | Once | Легкое шуршание рассыпаемой земли или песка. |
| **Полив** | **Процесс** | Loop | Журчание льющейся воды, звук впитывания влаги (всасывающий звук). |
| **Удобрение**| **Рассыпание** | Once | Сухой, сыпучий звук (как пересыпание гравия или песка). |
| **Удобрение**| **Органика** | Once | Влажный, чавкающий звук шлепка. |
| **Скверна** | **Появление** | Once | Тихий, зловещий шепот. |
| **Скверна** | **Заражение соседей**| Once | Шипение кислоты. |
| **Сбор** | **Руками (овощи)** | Once | Звук выдергивания корня из земли. |
| **Сбор** | **Серпом (злаки)** | Once | Резкий, свистящий звук рассекания воздуха. |

| Объект | Состояние | Визуализация | Примечание |
| :--- | :--- | :--- | :--- |
| **Грядка** | Сухая | Светло-коричневый/Серый спрайт грунта. | |
| **Грядка** | Политая | Темно-коричневый, блестящий спрайт (эффект влажности). | Постепенно светлеет по мере высыхания. |
| **Растение** | Мутация | Легкое неестественное свечение (например, красное или фиолетовое) вокруг стебля. | Указывает на алхимические свойства. |
| **Растение** | Скверна | Розовая гниль, споры и паутина. | |

## 5. Таблицы оборудования и инструментов

### Таблица 1: Инструменты фермера

| Инструмент | Функция | Описание |
| :--- | :--- | :--- |
| **Мотыга** | Вспашка | Превращает тайл грунта в грядку. |
| **Лопата** | Копка | Позволяет выкопать растение целиком (для пересадки) или уничтожить грядку. |
| **Лейка** | Полив | Вмещает жидкость (воду, кровь, алхимию). Обеспечивает точечный полив тайла. |
| **Серп** | Сбор (злаки/травы) | Увеличивает скорость сбора урожая. Повышает шанс подобрать больше. |
| **Нож** | Лечение | Позволяет точечно срезать пораженные скверной участки без уничтожения растения. |

# Библиотека растений

## 1. Огородные культуры
*Выращиваются на грядках. Подчиняются механике NPK. Требуют света (кроме специфических).*

| Категория | Растения | Потребление (N / P / K) | Игровые особенности |
| :--- | :--- | :--- | :--- |
| **Злаки** | Рожь, овес, пшеница, кукуруза | 3 / 2 / 2 (Истощают почву) | Требуют переработки. |
| **Бобовые**| Горох, соя, бобы | **-2** / 2 / 2 (Обогащают азотом) | Важнейший элемент севооборота. Сажаются для восстановления почвы после злаков и листовых. |
| **Корнеплоды** | Морковь, свекла, пастернак, картофель | 1 / 3 / 2 |  |
| **Листовые/стеблевые**| Капуста, лук, лен, хмель | 3 / 1 / 1 | Лен — техническая культура (ткань). Хмель — для алкоголя. |
| **Плодовые/ягоды** | Огурец, помидор, клубника, виноград | 2 / 2 / 3 | Истощают калий. Идеальны для засолки и создания вина. |

## 2. Дикорастущие
*Генерируются в мире. Не требуют ухода. Не истощают почву.*

| Категория | Растения | Среда обитания | Особенности |
| :--- | :--- | :--- | :--- |
| **Дикие плоды** | Черника, клюква, оливы | Леса, болота | Можно пересадить лопатой, но растут крайне медленно. Оливы дают масло. |
| **Луговые травы** | Клевер, одуванчик, ията, крапива | Открытые поля | Крапива используется как дешевый аналог ниток или сырье для грубых супов. |
| **Деревья** | Яблоня, груша | Леса, дикие сады | Многолетние. Урожай появляется циклично.|

## 3. Алхимия 
*Специфические растения с особыми свойствами.*

| Растение | Тип выращивания | Среда / Требования | Алхимические / Игровые особенности |
| :--- | :--- | :--- | :--- |
| **Полынь, зверобой, валериана, подорожник** | Дикорастущее | Лесные поляны | Базовые ингредиенты для обеззараживающих мазей, бинтов и успокоительных чаев. |
| **Мандрагора** | Огородное (сложное) | Высокое потребление P и K. Пока не расписано. | Сильнейший реагент. |
| **Белладонна, аконит** | Огородное (ядовитое) | Стандартная грядка | Ядовиты при употреблении в сыром виде. Используются для создания ядов. |
| **Могильный мох** | Дикорастущее | Кладбища | Появляется на могилах время от времени |
