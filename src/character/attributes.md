# Атрибуты
Тут должны быть атрибуты персонажа, выбираемые при старте. Пока чисто заглушка.

## Характеристики (1-10)

### 1. Сила
* **Влияет на:**
    * Урон в ближнем бою.
    * Возможность использовать тяжелое оружие (двуручные мечи, молоты).
    * Скорость добычи руды.

### 2. Ловкость
* **Влияет на:**
    * Скорость передвижения.
    * Шанс уворота в боевом режиме.
    * Скорость взаимодействия с мелкими объектами (бинтование).

### 3. Выносливость
* **Влияет на:**
    * Максимальное здоровье.
    * Сопротивление ядам и болезням.
    * Длительность спринта.

### 4. Воля
* **Влияет на:**
    * Сопротивление потери рассудка.
    * Эффективность магии.
    * Сопротивление ментальным атакам.
