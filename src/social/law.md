# Законы

## 1. Концепция
Законы задают правила поселения и создают игровые поводы для суда, наказаний, конфликтов и власти.

## 2. Категории нарушений
| Нарушение | Пример реакции |
| :--- | :--- |
| Мелкое | Штраф, предупреждение, общественная работа. |
| Среднее | Арест, конфискация, суд. |
| Тяжкое | Казнь, изгнание, долгий арест. |
| Религиозное | Отдельное решение церкви или власти. |

## 3. Открытые вопросы
1. Кто пишет законы?
2. Кто имеет право судить?
