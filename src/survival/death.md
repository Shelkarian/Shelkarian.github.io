# Смерть и возрождение

## 1. Концепция
Смерть должна быть сильным последствием, но не обязательно концом участия игрока в раунде.

## 2. Состояния
| Состояние | Описание |
| :--- | :--- |
| Критическое | Персонаж жив, но не может нормально действовать. |
| Смерть | Тело погибло. |
| Призрак | Игрок наблюдает или ждет роли. |
| Возрождение | Возврат через редкую механику, если она разрешена. |

3. Что происходит с памятью после возвращения?
