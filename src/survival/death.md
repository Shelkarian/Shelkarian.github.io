# Критическое состояние, смерть и возвращение

## Состояния персонажа

| Состояние | Тело | Управление игрока | Переход дальше |
| :--- | :--- | :--- | :--- |
| Жив | Функционирует | Полное с учетом эффектов | В критическое при достижении порога здоровья |
| Критическое | Живо, без сознания | Нет действий телом; доступен интерфейс ожидания | В стабилизацию, жизнь или смерть |
| Мертв | Труп остается в мире | Управление телом отсутствует | В призрака; редко — воскрешение |
| Призрак | Не имеет физического тела | Наблюдение и разрешенные призрачные роли | В новую разрешенную роль или воскрешенное тело |
| Воскрешен | Исходное тело снова живо | Возвращается после короткого оглушения | Дальше подчиняется обычным состояниям |

## Критическое состояние

Критическое состояние начинается, когда суммарное здоровье достигает критического порога, но условие мгновенной смерти не выполнено.

| Параметр | Значение |
| :--- | :--- |
| Критический порог здоровья | `0` |
| Порог смерти от общего урона | `−100` |
| Базовое время до смерти без помощи | 5 минут |
| Проверка ухудшения | Каждые 10 секунд |
| Самопроизвольное восстановление | Не предусмотрено |
| Перемещение и действия | Недоступны |

Пока персонаж критический, продолжающиеся источники урона действуют. Кровотечение, удушье, яд, огонь и повреждающая среда могут довести здоровье до `−100` раньше базового таймера. Метаболизм в этом состоянии приостановлен.

**Стабилизация** останавливает базовый таймер смерти, но не возвращает сознание и не отменяет отдельные источники урона. Для выхода в состояние «жив» здоровье должно стать выше `0`, а причина потери сознания должна быть устранена.

## Смерть

Смерть наступает при любом из условий:

- здоровье достигло `−100`;
- истекли 5 минут критического состояния без стабилизации;
- применено явно обозначенное мгновенно смертельное повреждение;
- тело находится в состоянии, несовместимом с жизнью по правилам вида или сценария.

При смерти фиксируются время, причина и ответственный источник, если он известен системе. Тело и его содержимое остаются в мире. Личные цели, предметы и должностные полномочия не переходят призраку.

## Призрак

После смерти игрок может стать призраком сразу либо оставаться в интерфейсе тела в ожидании помощи. Переход в призрака не запрещает редкое воскрешение.

| Возможность | Правило |
| :--- | :--- |
| Наблюдение | Можно следовать за персонажами и точками интереса |
| Общение | Только канал призраков; живые не слышат его без отдельной игровой механики |
| Взаимодействие с миром | Запрещено, кроме явно разрешенных призрачных механик |
| Передача информации живым | Запрещена без игровой способности с наблюдаемыми ограничениями |
| Новая роль | Разрешена правилами режима; после принятия возврат в старое тело запрещен |
| Воскрешение | Игрок получает запрос на возврат; отказ оставляет тело мертвым |

Призрак не видит скрытую информацию, которую режим специально защищает от наблюдателей, и не может использовать сведения призрака после получения новой живой роли.

## Редкое воскрешение

Воскрешение — сценарная, дорогая и прерываемая процедура, а не штатная медицинская услуга.

| Условие | Требование |
| :--- | :--- |
| Тело | Существует, опознано и не имеет состояния, явно запрещающего воскрешение |
| Срок после смерти | Не более 15 минут |
| Душа игрока | Не заняла новую роль и согласилась вернуться |
| Исполнитель | Жив, дееспособен и имеет доступ к разрешенному сценарием ритуалу или артефакту |
| Цена | Один невосполнимый в пределах раунда компонент на попытку |
| Время процедуры | 3 минуты непрерывного проведения |
| Прерывание | Перемещение тела, потеря исполнителя, разрушение компонента или выход из зоны ритуала сбрасывает попытку |
| Лимит | Не более одного успешного воскрешения одного персонажа за раунд |

Компонент расходуется при начале процедуры. Неудача не возвращает его. Ритуал создает заметные звук, свет или иной признак, позволяющий другим игрокам вмешаться.

Успешно воскрешенный персонаж возвращается с `10` здоровья, `20` питания и `20` гидратации, получает оглушение на 30 секунд и сохраняет память до момента смерти. Он не получает информацию, увиденную в режиме призрака. Повреждения и болезни тела сохраняются, кроме минимально исправленных процедурой состояний, без устранения которых жизнь невозможна.

## Особые ситуации

| Ситуация | Правило |
| :--- | :--- |
| Лечение и урон приходят в один тик | Сначала суммируются изменения здоровья; мгновенно смертельное условие проверяется отдельно; состояние определяется по итогу тика |
| Таймер критического состояния истекает одновременно со стабилизацией | Завершенная стабилизация имеет приоритет, если подтверждена в тот же тик |
| Тело уничтожено во время ритуала | Ритуал провален, компонент не возвращается |
| Игрок отключен при запросе воскрешения | Запрос ожидает до конца допустимых 15 минут; без согласия воскрешение не завершается |
| Игрок уже занял новую живую роль | Возврат в прежнее тело запрещен |
| Призрак отказался от воскрешения | Тело остается мертвым; повторный запрос возможен в пределах срока и лимита ресурсов |
| Раунд перешел в эпилог | Начатый ритуал можно завершить, новый начинать нельзя |
| Идет проверка досрочного падения | Валидный активный ритуал блокирует подтверждение, пока не завершится или не будет прерван |

## Ожидаемое поведение

- При здоровье `0` персонаж переходит в критическое состояние, а при `−100` умирает.
- Без помощи критическое состояние заканчивается смертью через 5 минут активного времени.
- Стабилизация останавливает базовый таймер, но не лечит продолжающийся урон.
- Призрак не взаимодействует с обычным миром и не сообщает живым скрытую информацию без отдельной механики.
- Воскрешение требует тела, согласия игрока, трех минут, невосполнимого компонента и укладывается в 15 минут после смерти.
- Один персонаж нельзя успешно воскресить более одного раза за раунд.
- Возвращенный персонаж не сохраняет знания, полученные после перехода в призрака.
