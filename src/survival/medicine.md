# Медицина и ранения

## 1. Концепция
Медицина отвечает за диагностику, лечение ран, остановку кровотечения, болезни и восстановление после боя.

## 2. Типы проблем
| Проблема | Лечение |
| :--- | :--- |
| Кровотечение | Бинт, давление, шов. |
| Перелом | Шина, покой, хирургия. |
| Ожог | Мазь, охлаждение. |
| Отравление | Противоядие, промывание. |
| Болезнь | Диагностика, лекарство, карантин. |

## 3. Открытые вопросы
1. Нужна ли хирургия на старте?
2. Как игрок видит тип раны?
3. Может ли неправильное лечение ухудшить состояние?
