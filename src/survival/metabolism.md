# Метаболизм

## 1. Концепция
Метаболизм описывает голод, жажду, сытость, усталость и влияние еды на персонажа.

## 2. Параметры
| Параметр | Эффект |
| :--- | :--- |
| Голод | Снижает восстановление и выносливость. |
| Жажда | Вредит при долгом игнорировании. |
| Сытость | Дает время до следующего голода. |
| Качество еды | Может давать бонусы или штрафы. |
