# Метапрогрессия

## 1. Концепция
Метапрогрессия описывает, что сохраняется между раундами и не ломает честность конкретного раунда.

## 2. Можно сохранять
| Элемент | Почему допустимо |
| :--- | :--- |
| Открытые записи лора | Не дает прямой силы. |
| Косметика | Не влияет на баланс. |
| Репутация аккаунта | Помогает доступам и доверию. |
| Энциклопедия рецептов | Если рецепты все равно можно узнать в раунде. |

## 3. Нельзя сохранять без решения
1. Боевые бонусы.
2. Деньги.
3. Редкие предметы.
4. Прямой доступ к сильным ролям.
