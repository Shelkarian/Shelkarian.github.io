# Концовки

## 1. Концепция
Концовка описывает, чем закончился раунд и какие последствия получили игроки, поселение или мир.

## 2. Типы концовок
| Тип | Условие |
| :--- | :--- |
| Выживание | Поселение пережило угрозу. |
| Падение | Ключевые структуры разрушены. |
| Победа фракции | Одна группа выполнила цель. |
| Катастрофа | Мир получил серьезное последствие. |
