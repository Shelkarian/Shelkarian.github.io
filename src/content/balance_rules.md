# Стандарты баланса

## 1. Назначение
Страница фиксирует общие правила баланса для предметов, профессий, рецептов, существ и наград.

## 2. Базовые принципы
| Правило | Описание |
| :--- | :--- |
| Нет бесплатной силы | Сильный эффект требует ресурса, времени, риска или редкости. |
| Контр-игра | У опасной механики должен быть способ защиты. |
| Читаемость | Игрок должен понимать, почему проиграл или победил. |
| Роль важнее цифры | Предмет должен иметь назначение, а не просто больший урон. |

## 3. Проверка контента
1. Что игрок получает?
2. Чем он платит?
3. Кто страдает от этой механики?
4. Есть ли контр-действие?
5. Можно ли это абузить массово?
