# Книга рецептов

## 1. Назначение
Книга рецептов хранит список крафтов, готовки, алхимии, кузнечного дела и строительства.

## 2. Формат рецепта
| Поле | Описание |
| :--- | :--- |
| Результат | Что создается. |
| Ингредиенты | Что требуется. |
| Инструмент | Чем выполняется работа. |
| Станция | Где выполняется работа. |
| Время | Сколько длится процесс. |
| Навык | Что влияет на качество. |

## 3. Пример
| Результат | Ингредиенты | Инструмент | Станция |
| :--- | :--- | :--- | :--- |
| Жареное мясо | Сырое мясо, жир | Сковорода | Плита/огонь |
